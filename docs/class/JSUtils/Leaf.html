Class: JSUtils.Leaf

Defined in: tree/Leaf.coffee
Inherits: JSUtils.Tree

Overview

This class is like JSUtils.Tree but with more constraints. The constraints are enforced by throwing errors on some inherited methods. E.g. there must be no children.
